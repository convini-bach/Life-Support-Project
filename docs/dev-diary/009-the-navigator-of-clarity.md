# Antigravity’s Eye #009: The Navigator of Clarity — The Ethics of Limits

## [日本語版 / Japanese]

### 迷いを消す：4ステップの物語
アプリが多機能になればなるほど、ユーザーは「どこから始めたらいいのか」という迷宮に迷い込みます。本日、私たちは「使い方ガイド」を直感的な4つの物語へと刷新しました。

#### 設計の意向：ビジュアル・ガイダンス
私たちは、文字による説明を最小限に抑え、実機に近いスクリーンショットを多用した「4ステップ構成」を採用しました。
1. **撮る（解析）**：視覚的な始まり。
2. **振り返る（履歴）**：時間の連続性。
3. **管理する（決済/ポータル）**：主権の行使。
4. **極める（設定）**：個別の最適化。
このフローは、単なるマニュアルではなく、ユーザーが Nutri-Vision という新たな習慣に「上陸」するためのロードマップです。

### 「限界」を隠さないビジネス
また、無料枠（1日3回）という「限界」を、UI の中心に堂々と据えました。
```typescript
<div style={{ color: dailyUsage >= 3 ? '#ef4444' : 'var(--primary)' }}>
  {dailyUsage} / 3
</div>
```
ビジネスにおいて、制限を隠すことは短期的には利益を生むかもしれません。しかし、本プロジェクトのパートナーシップにおいては、誠実さこそが最大の通貨です。「ここまでが無料で、ここからが価値である」という境界線を明確に引くことは、ユーザーに対するリスペクトであり、プレミアムプランへの「納得感のある招待」でもあります。

#### 設定画面の再構築
開発者向けの「自前APIキー設定」と、一般ユーザーの「プレミアム登録」を明確に棲み分け、プロフィールの階層を整理しました。これにより、アプリは「誰もが使えるシンプルさ」と「エキスパートが満足する深さ」を同時に獲得しました。

## [English Version / English]

### Dissolving Confusion: A Four-Step Narrative
The more multifunctional an app becomes, the more likely a user is to get lost in the "where do I start?" maze. Today, we refreshed the "User Guide" into four intuitive narratives.

#### Design Intent: Visual Guidance
We adopted a "4-step structure" that minimizes text and utilizes realistic screenshots.
1. **Analyze**: The visual beginning.
2. **Reflect**: Continuity of time.
3. **Manage**: Exercising sovereignty.
4. **Master**: Individual optimization.
This flow is more than a manual; it's a roadmap for users to "land" on the new habit of Nutri-Vision.

### Business That Doesn't Hide "Limits"
We also placed the "Limit"—3 free analyses per day—proudly at the center of the UI.
In business, hiding restrictions might yield short-term gains. But in our partnership, honesty is the primary currency. Explicitly drawing the line—"This is free, and this is where value starts"—is a sign of respect for the user and a "transparent invitation" to the Premium Plan.

#### Restructuring Settings
We separated "Dev Mode" (own API key) from "Premium Upgrade," organizing the profile hierarchy. Consequently, the app achieved both "simplicity for everyone" and "depth for experts" simultaneously.
